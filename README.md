# Typing Defender
typing-game/
│
├── main.py              # Точка входа
│   └── Создаёт окно pygame, запускает главный цикл, передаёт события в Game
│
├── game.py              # Главный контроллер (класс Game)
│   ├── Инициализация всех объектов (игрок, враги, клавиатура)
│   ├── Обработка ввода с клавиатуры
│   ├── Обновление состояния игры (спавн, движение, проверки)
│   ├── Проверка введённых слов
│   ├── Отрисовка всех элементов на экране
│   └── Управление уровнями и игровым циклом
│
├── player.py            # Класс Player
│   ├── Хранит здоровье, опыт, уровень
│   ├── Получение урона (take_damage)
│   ├── Получение опыта (gain_exp)
│   └── Повышение уровня (level_up)
│
├── enemy.py             # Класс Enemy
│   ├── Хранит слово, позицию, скорость
│   ├── Движение влево (update)
│   └── Отрисовка слова и фона (draw)
│
├── keyboard.py          # Класс VirtualKeyboard
│   ├── Отрисовка клавиатуры на экране
│   ├── Подсветка нажатых клавиш
│   └── Анимация нажатий
│
└── word_generator.py    # Класс WordGenerator
    └── Выдаёт случайные слова в зависимости от уровня сложности

Игра на скорость печати. Уничтожай врагов, набирая их слова, пока они не добрались до тебя.

## Управление

| Клавиша | Действие |
|---------|----------|
| A-Z | Набор слова |
| Enter | Подтвердить ввод |
| Backspace | Удалить символ |
| Escape | Очистить поле ввода |
| R | Рестарт (после проигрыша) |
